# /add-scene

バズmodeの背景プール（BACKGROUNDS）に新しい背景シーンを追加する。

## 引数
`$ARGUMENTS` — 追加したい背景の説明（例: "桜の木の下、春の公園"）

## 実行手順

1. `agents/image_agent.py` を読み込む
2. `BACKGROUNDS` リストを見つける
3. `$ARGUMENTS` の背景を適切な形式でリストに追加する
   - 形式: `"[English description]",  # [日本語メモ]`
   - 英語で書くこと（プロンプトが英語のため）
   - 季節感がある場合は季節を英語で記載（spring/summer/autumn/winter）
   - 既存の背景と重複しないか確認する
4. 追加したことをユーザーに報告する（何番目に追加されたか）

## 注意
- 背景プールのwindow（重複防止期間）は5。追加後にリストが5以下にならないよう確認。
- 背景は「室内」か「屋外」かを意識し、バランスよく維持する。
