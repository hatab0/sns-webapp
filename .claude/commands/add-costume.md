# /add-costume

バズmodeの衣装プールに新しい衣装を追加する。

## 引数
`$ARGUMENTS` — 追加したい衣装の説明（例: "ライトブルーのニットセーター、袖口に白いレースあり"）

## 実行手順

1. `agents/image_agent.py` を読み込む
2. `COSTUMES` リストを見つける
3. `$ARGUMENTS` の衣装を適切な形式でリストに追加する
   - 形式: `"[English description]",  # [日本語メモ]`
   - 英語で書くこと（プロンプトが英語のため）
   - 既存の衣装と重複しないか確認する
4. 追加したことをユーザーに報告する（何番目に追加されたか）

## 注意
- 衣装プールのwindow（重複防止期間）は7。追加後にリストが7以下にならないよう確認。
- 帽子・ニット帽は衣装ではなく別のアクセサリーとして扱う。
